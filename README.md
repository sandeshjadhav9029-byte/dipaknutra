# Dipak Nutra E-Commerce Website

A Next.js e-commerce website for Dry Fruits & Natural Products.

## Pages
- Home
- Shop (Products: W320, W240, Flavored Cashews, etc.)
- Product Detail
- Cart
- Checkout
- About Us
- Contact
- FAQ
- Shipping & Returns
- Admin Panel

## Tech Stack
- Next.js 16
- Tailwind CSS
- Zustand (state management)
- Lucide React (icons)

## Getting Started

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Deployment

Deploy to Vercel:
1. Push to GitHub
2. Import in Vercel dashboard
3. Deploy

Or use Netlify: https://app.netlify.com
